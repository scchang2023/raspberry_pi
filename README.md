# raspberry_pi

## 在 raspberry pi 上安裝作業系統

- [Raspberry Pi OS](https://www.raspberrypi.com/software/)：到官網下載

![alt text](image.png)

下載完成後，為一個imager_1.8.5.exe的執行檔。

- 執行 imager_1.8.5
- 選擇 raspberry pi device
- 選擇操作系統
- 選擇儲存卡
- 輯輯設定
  - 主機名：raspberrypiSCC.local
  - username：pi
  - password：raspberry
  - WIFI 名稱： LINKOU203-4F-5G
  - WIFI 密碼： xxx
- 開始燒錄
